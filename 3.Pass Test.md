## 1. Overall pass rule

A phase is **PASS** only if:

* all **WI hard gates** pass
* at least **90%** of that phase’s quantitative checks pass
* there are **0 critical failures**

A phase is **CONDITIONAL PASS** only if:

* all **WI hard gates** pass
* **80% to 89%** of that phase’s quantitative checks pass
* there are **0 critical failures**

A phase is **FAIL** if:

* any WI hard gate fails
* any critical failure occurs
* or fewer than **80%** of the quantitative checks pass

* heatmap leaves stale materials after clear/reload
* debug filters show wrong-channel or wrong-severity data
* any red console error during normal smoke flow

## 3. Common smoke flow

Use this smoke flow for every phase before marking PASS.

1. Open app
2. Open `Advanced GLB Viewer`
3. Open `Pro GLB Viewer`
4. Switch left panel between `Import`, `Settings`, `Debug`
5. Load one valid GLB
6. Use camera controls: orbit, zoom, pan, fit, reset
7. If implemented in that phase, test property/section/measure/heatmap/debug features
8. Switch away to another tab
9. Return to `Pro GLB Viewer`
10. Repeat cycle **5 times**

Pass condition: full smoke flow succeeds **5/5** times. The WI explicitly expects reuse of existing viewer modules, inline settings, structured debug, and lifecycle cleanup.
