## 0. Codebase Assessment — What Exists, What Is Correct, What Needs Work

### 0.1 What is already correct and must not be changed

| Module | Location | Status |
|--------|----------|--------|

| State singleton | `viewer/core/state.js` | ✅ Use as-is. `state.parsed` holds accdb/xml/pdf data |
| PCF parser pipeline | `viewer/js/pcf2glb/pcf/` | ✅ `parsePcfText` → `normalizePcfModel` chain works |
| GLB builder | `viewer/js/pcf2glb/glb/` | ✅ `buildExportScene` + `exportSceneToGLB` are functional |
| State-to-PCF bridge | `viewer/utils/accdb-to-pcf.js` | ✅ `buildUniversalCSV` + `normalizeToPCF` are the connectors |
| Component panel model | `viewer/viewer3d/component-panel-model.js` | ✅ Section-based model is the right pattern; extend it |
| Advanced GLB Viewer | `viewer/js/pcf2glb/advanced/` | ✅ All `create*` modules are well-separated; reuse them |
| Simple GLB Exporter | `viewer/js/pcf2glb/ui/PcfGlbExporterPanel.js` | ✅ Import source #2 (app state) already wired here |
| Event bus | `viewer/core/event-bus.js` | ✅ Use `on`/`emit` for cross-tab communication |
| Logger | `viewer/core/logger.js` | ✅ `addTraceEvent` exists; extend the debug tab to consume it |

### 0.2 Identified gaps and flaws in the current codebase

**Gap 1 — Advanced GLB Viewer has no PCF file import**  
`AdvancedGlbViewerPanel.js` only accepts `.glb/.gltf` files. It cannot ingest `.pcf` files
directly. The "Simple GLB Exporter" tab has PCF → GLB pipeline but no professional viewer.
These capabilities must be unified in the new Pro GLB Viewer tab.

**Gap 2 — Property panel is hard-coded, not component-type-aware in Advanced Viewer**  
`createPropertyPanel.js` renders a generic raw-metadata table. The existing `component-panel-model.js`
(used by `viewer3d-tab.js`) already has a section-based, type-aware model but is NOT wired into
the advanced viewer. The new Pro GLB Viewer must use and extend this model.

**Gap 3 — Debug panel in Advanced Viewer is only a log viewer with no structured channels**  
`createDebugPanel.js` renders a flat array of log entries. During development, this is insufficient.
The new panel needs: channel filtering, severity filtering, JSON expand/collapse, performance counters,
and an export-to-clipboard function.

**Gap 4 — No settings/config sub-tab inside the GLB viewer panels**  
The 3D Viewer tab has a `Config` button that jumps to a separate `config` tab. The Pro GLB Viewer
must contain its own inline settings sub-tab (camera, rendering, property panel fields, debug verbosity).

**Gap 5 — No vertex snapping or measurement tools in either existing GLB panel**  
Neither `AdvancedGlbViewerPanel` nor `PcfGlbExporterPanel` has measurement tools.

**Gap 6 — `currentFile` URL is not revoked on panel teardown**  
`AdvancedGlbViewerPanel.js` calls `URL.createObjectURL` but has no cleanup path if the tab changes.
This leaks object URLs. The Pro GLB Viewer must clean up in a `destroy()` lifecycle call.

**Gap 7 — `app.js` tab change does not call `destroy()` on outgoing tab**  
The tab router calls `render(container)` on mount but has no mechanism to call cleanup on unmount.
This needs a minor enhancement to `_switchTab`.

```javascript
// ENHANCEMENT: give the tab router a destroy hook so panels can clean up
let _activeDestroyFn = null;

function _switchTab(tabId) {
  // Call previous tab's destroy if it exposed one
  if (typeof _activeDestroyFn === 'function') {
    try { _activeDestroyFn(); } catch(e) { console.warn('[app] destroy error:', e); }
    _activeDestroyFn = null;
  }

  state.activeTab = tabId;
  const content = document.getElementById('tab-content');
  const tab = TABS.find(t => t.id === tabId);
  if (!tab || !content) return;

  content.innerHTML = '';
  const result = tab.render(content);
  // If render returns { destroy }, store it
  if (result && typeof result.destroy === 'function') {
    _activeDestroyFn = result.destroy;
  }
  emit('tab-changed', tabId);
}
```

### 5.3 `Proglb_DebugConsole.js` — Structured Debug Console

```javascript
// viewer/js/pcf2glb/debug/Proglb_DebugConsole.js
import { CH, LEVEL, LEVEL_NAMES } from './Proglb_DebugChannels.js';

const MAX_ENTRIES = 2000;

export function createDebugConsole(containerEl, opts = {}) {
  // opts.minLevel: default TRACE during dev, INFO in prod
  // opts.enabledChannels: null = all, or Set of channel names

  const entries = [];
  let minLevel = opts.minLevel ?? LEVEL.TRACE;
  let enabledChannels = opts.enabledChannels ?? null; // null = all
  let dirty = false;
  let rafId = null;

  // ── Internal logger ────────────────────────────────────────────────────────
  function log(channel, level, message, payload) {
    // Always push to entries array (full history)
    entries.push({
      ts: Date.now(),
      channel,
      level,
      message,
      payload: payload ?? null,
    });
    if (entries.length > MAX_ENTRIES) entries.shift();

    // Also mirror to browser console during development (remove in prod)
    const fn = level >= LEVEL.ERROR ? console.error
             : level >= LEVEL.WARN  ? console.warn
             : level >= LEVEL.INFO  ? console.info
             : console.debug;
    fn(`[${channel}] ${message}`, payload ?? '');

    dirty = true;
    if (!rafId) rafId = requestAnimationFrame(_flush);
  }

  // ── Convenience wrappers ───────────────────────────────────────────────────
  const logger = {
    trace: (ch, msg, p) => log(ch, LEVEL.TRACE, msg, p),
    debug: (ch, msg, p) => log(ch, LEVEL.DEBUG, msg, p),
    info:  (ch, msg, p) => log(ch, LEVEL.INFO,  msg, p),
    warn:  (ch, msg, p) => log(ch, LEVEL.WARN,  msg, p),
    error: (ch, msg, p) => log(ch, LEVEL.ERROR, msg, p),
    log,
    setMinLevel: (l) => { minLevel = l; _flush(); },
    setChannel:  (ch, on) => {
      if (!enabledChannels) enabledChannels = new Set(Object.values(CH));
      if (on) enabledChannels.add(ch); else enabledChannels.delete(ch);
      _flush();
    },
    clearChannelFilter: () => { enabledChannels = null; _flush(); },
    getEntries: () => [...entries],
    exportToClipboard: () => {
      const text = entries.map(e =>
        `[${new Date(e.ts).toISOString()}][${LEVEL_NAMES[e.level]}][${e.channel}] ${e.message}` +
        (e.payload ? '\n  ' + JSON.stringify(e.payload, null, 2) : '')
      ).join('\n');
      navigator.clipboard.writeText(text).then(
        () => log(CH.SETTINGS, LEVEL.INFO, 'Debug log copied to clipboard'),
        (err) => log(CH.SETTINGS, LEVEL.WARN, 'Clipboard write failed', { err: err.message })
      );
    },
    destroy: () => { if (rafId) { cancelAnimationFrame(rafId); rafId = null; } },
  };

  // ── Render to DOM ──────────────────────────────────────────────────────────
  function _flush() {
    rafId = null;
    dirty = false;
    if (!containerEl) return;

    const filtered = entries.filter(e =>
      e.level >= minLevel &&
      (!enabledChannels || enabledChannels.has(e.channel))
    );

    // Only render last 200 for performance
    const visible = filtered.slice(-200);
    containerEl.innerHTML = visible.map(e => {
      const levelName = LEVEL_NAMES[e.level];
      const color = e.level >= LEVEL.ERROR ? '#ff6b6b'
                  : e.level >= LEVEL.WARN  ? '#ffa94d'
                  : e.level >= LEVEL.INFO  ? '#74c0fc'
                  : e.level >= LEVEL.DEBUG ? '#a9e34b'
                  : '#666';
      const ts = new Date(e.ts).toISOString().slice(11, 23); // HH:MM:SS.mmm
      const payloadStr = e.payload
        ? `<div style="color:#666;padding-left:20px;word-break:break-all;">${
            _safeJson(e.payload)
          }</div>`
        : '';
      return `<div style="margin-bottom:3px;padding-bottom:3px;border-bottom:1px solid #1a1a2e;">
        <span style="color:#555;font-size:10px;">${ts}</span>
        <span style="color:${color};font-weight:bold;margin:0 4px;">[${levelName}]</span>
        <span style="color:#8899aa;">[${e.channel}]</span>
        <span>${_esc(e.message)}</span>
        ${payloadStr}
      </div>`;
    }).join('');
    containerEl.scrollTop = containerEl.scrollHeight;
  }

  function _safeJson(v) {
    try { return _esc(JSON.stringify(v, null, 1)); }
    catch { return String(v); }
  }
  function _esc(s) {
    return String(s)
      .replace(/&/g, '&amp;')
      .replace(/</g, '&lt;')
      .replace(/>/g, '&gt;');
  }

  return logger;
}
```

```javascript
    maxRawRows: 30,         // cap raw attributes shown
    floatDecimalPlaces: 3,
  },
};

export function loadSettings() {
  try {
    const raw = localStorage.getItem(STORAGE_KEY);
    if (!raw) return _deepClone(DEFAULT_SETTINGS);
    return _deepMerge(_deepClone(DEFAULT_SETTINGS), JSON.parse(raw));
  } catch (e) {
    console.warn('[Proglb_ViewerSettingsStore] load failed, using defaults', e);
    return _deepClone(DEFAULT_SETTINGS);
  }
}

export function saveSettings(settings) {
  try {
    localStorage.setItem(STORAGE_KEY, JSON.stringify(settings));
  } catch (e) {
    console.warn('[Proglb_ViewerSettingsStore] save failed', e);
  }
}

export function resetSettings() {
  localStorage.removeItem(STORAGE_KEY);
  return _deepClone(DEFAULT_SETTINGS);
}

function _deepClone(v) { return JSON.parse(JSON.stringify(v)); }

function _deepMerge(target, source) {
  for (const key of Object.keys(source)) {
    if (source[key] && typeof source[key] === 'object' && !Array.isArray(source[key])) {
      target[key] = _deepMerge(target[key] ?? {}, source[key]);
    } else {
      target[key] = source[key];
    }
  }
  return target;
}
```

### 5.6 `Proglb_PropertyPanelRenderer.js` — Type-Aware Configurable Panel

```javascript
// viewer/js/pcf2glb/property/Proglb_PropertyPanelRenderer.js
import { getSchemaForType } from './Proglb_ComponentTypeSchemas.js';
import { CH } from '../debug/Proglb_DebugChannels.js';

/**
 * createPropertyPanelRenderer(panelEl, contentEl, settings, logger)
 *
 * settings: Proglb_ViewerSettingsStore settings.propertyPanel subtree
 * logger:   Proglb_DebugConsole logger instance
 */
export function createPropertyPanelRenderer(panelEl, contentEl, settings, logger) {
  const pp = settings?.propertyPanel ?? {};
  const dp = settings?.debug ?? {};
  const decimalPlaces = pp.floatDecimalPlaces ?? 3;

  function _resolveField(field, attrs) {
    // Try explicit key
    if (field.key) {
      const v = attrs[field.key];
      if (v !== null && v !== undefined && v !== '') return String(v);
    }
    // Try fallback key list
    if (field.keys) {
      for (const k of field.keys) {
        const v = attrs[k];
        if (v !== null && v !== undefined && v !== '') return String(v);
      }
    }
    return null;
  }

  function _formatValue(raw, field) {
    if (raw === null || raw === undefined) return '—';
    if (field.format === 'float') {
      const n = parseFloat(raw);
      return isNaN(n) ? raw : n.toFixed(decimalPlaces) + (field.unit ? ' ' + field.unit : '');
    }
    if (field.format === 'int') {
      const n = parseInt(raw, 10);
      return isNaN(n) ? raw : n.toString() + (field.unit ? ' ' + field.unit : '');
    }
    return raw + (field.unit ? ' ' + field.unit : '');
  }

  /**
   * Build a virtual attributes map from a sceneIndex item.
   * Adds derived keys like _ep1_fmt, _length, etc.
   */
  function _buildAttrs(item) {
    const attrs = { ...(item.rawMeta ?? {}), ...(item.object3D?.userData ?? {}) };

    // Derived geometry keys
    const p1 = item.ep1 ?? null;
    const p2 = item.ep2 ?? null;
    const centre = item.centrePoint ?? null;
    const bore = item.bore ?? 0;

    if (p1) attrs._ep1_fmt = `${_f(p1.x)}, ${_f(p1.y)}, ${_f(p1.z)}`;
    if (p2) attrs._ep2_fmt = `${_f(p2.x)}, ${_f(p2.y)}, ${_f(p2.z)}`;
    if (centre) attrs._centre_fmt = `${_f(centre.x)}, ${_f(centre.y)}, ${_f(centre.z)}`;
    if (item.coOrds) attrs._coords_fmt = `${_f(item.coOrds.x)}, ${_f(item.coOrds.y)}, ${_f(item.coOrds.z)}`;
    attrs._bore = bore.toFixed(decimalPlaces);

    if (p1 && p2) {
      const dx = p2.x - p1.x, dy = p2.y - p1.y, dz = p2.z - p1.z;
      attrs._length = Math.hypot(dx, dy, dz).toFixed(decimalPlaces);
    }

    // Support block inference
    const merged = `${attrs.SUPPORT_NAME ?? ''} ${attrs.SUPPORT_TAG ?? ''}`.toUpperCase();
    const m = merged.match(/\bCA\d+\b/);
    attrs._inferred_block = m ? m[0] : '—';

    return attrs;
  }

  function _f(v) { return Number(v ?? 0).toFixed(decimalPlaces); }

  function _renderSection(section, attrs) {
    // Check show-section setting
    const sectionFlag = {
      common:    pp.showCommonSection,
      geometry:  pp.showGeometrySection,
      process:   pp.showProcessSection,
      support:   pp.showSupportSection,
      raw:       pp.showRawSection,
    }[section.id];
    if (sectionFlag === false) return '';

    let rows = [];

    if (section.rawDynamic) {
      // Raw fallback: dump all attributes not already shown
      const maxRows = pp.maxRawRows ?? 30;
      rows = Object.entries(attrs)
        .filter(([k]) => !k.startsWith('_'))
        .slice(0, maxRows)
        .map(([k, v]) => ({ label: k, value: String(v ?? '—') }));
    } else {
      for (const field of section.fields) {
        const raw = _resolveField(field, attrs);
        if (raw === null && field.optional) continue; // hide optional empty fields
        const value = _formatValue(raw, field);
        rows.push({ label: field.label, value });
      }
    }

    if (rows.length === 0) return '';

    return `
      <div style="margin-bottom:10px;">
        <div style="font-size:10px;font-weight:bold;color:#88aacc;text-transform:uppercase;
                    letter-spacing:0.08em;border-bottom:1px solid #334;padding-bottom:3px;margin-bottom:5px;">
          ${_esc(section.title)}
        </div>
        <table style="width:100%;border-collapse:collapse;font-family:monospace;font-size:11px;">
          <tbody>
            ${rows.map(row => `
              <tr>
                <td style="padding:3px 4px;border-bottom:1px solid #2a2a3a;color:#889aaa;width:45%;
                           word-break:break-word;">${_esc(row.label)}</td>
                <td style="padding:3px 4px;border-bottom:1px solid #2a2a3a;color:#e0e0e0;
                           word-break:break-word;">${_esc(row.value)}</td>
              </tr>
            `).join('')}
          </tbody>
        </table>
      </div>
    `;
  }

  function _esc(s) {
    return String(s).replace(/&/g,'&amp;').replace(/</g,'&lt;').replace(/>/g,'&gt;');
  }

  return {
    show: (item) => {
      if (!item) { panelEl.style.display = 'none'; return; }

      const type = String(item.type ?? item.rawMeta?.pcfType ?? '').toUpperCase();
      const schema = getSchemaForType(type);
      const attrs = _buildAttrs(item);

      logger?.debug(CH.SELECTION, `PropertyPanel.show: type="${type}" id="${item.id}"`, {
        schemaType: schema.type,
        attrKeys: Object.keys(attrs),
      });

      const title = `${type || 'COMPONENT'} : ${item.id ?? '—'}`;
      let html = `
        <div style="font-size:12px;font-weight:bold;color:#cce;margin-bottom:10px;
                    padding-bottom:8px;border-bottom:1px solid #334;">
          ${_esc(title)}
        </div>
      `;
      for (const section of schema.sections) {
        html += _renderSection(section, attrs);
      }

      contentEl.innerHTML = html;
      panelEl.style.display = 'block';
    },
    hide: () => { panelEl.style.display = 'none'; },
  };
}
```

### 5.7 `Proglb_StateImportAdapter.js` — App State → GLB

```javascript
// viewer/js/pcf2glb/import/Proglb_StateImportAdapter.js
import { state } from '../../../core/state.js';
import { buildUniversalCSV, normalizeToPCF } from '../../../utils/accdb-to-pcf.js';
import { buildExportScene } from '../glb/buildExportScene.js';
import { exportSceneToGLB } from '../glb/exportSceneToGLB.js';
import { timeStep } from '../pipeline/timeStep.js';
import { CH } from '../debug/Proglb_DebugChannels.js';

function _pt(v) {
  if (!v) return null;
  return { x: Number(v.x ?? 0), y: Number(v.y ?? 0), z: Number(v.z ?? 0), bore: Number(v.bore ?? 0) };
}

function _resolveNodePositions(csvRows) {
  const nodePos = new Map();
  if (!csvRows?.length) return nodePos;
  const first = csvRows[0];
  if (first?.FROM_NODE !== undefined) nodePos.set(first.FROM_NODE, { x: 0, y: 0, z: 0 });
  let progress = true, guard = 0;
  while (progress && guard < csvRows.length * 4) {
    guard++; progress = false;
    for (const row of csvRows) {
      const a = nodePos.get(row.FROM_NODE), b = nodePos.get(row.TO_NODE);
      const dx = Number(row.DELTA_X || 0), dy = Number(row.DELTA_Y || 0), dz = Number(row.DELTA_Z || 0);
      if (a && !b) { nodePos.set(row.TO_NODE, { x: a.x+dx, y: a.y+dy, z: a.z+dz }); progress = true; }
      else if (!a && b) { nodePos.set(row.FROM_NODE, { x: b.x-dx, y: b.y-dy, z: b.z-dz }); progress = true; }
    }
  }
  return nodePos;
}

/**
 * Returns { blob, componentCount } or throws.
 * @param {object} logger - Proglb_DebugConsole logger
 * @param {object} pipelineLog - legacy createLogger() for timeStep
 */
export async function importFromAppState(logger, pipelineLog) {
  const parsed = state.parsed;
  if (!parsed?.elements?.length) {
    logger.warn(CH.IMPORT, 'importFromAppState: state.parsed has no elements');
    throw new Error('No parsed data in app state. Load an ACCDB/XML/PDF file first.');
  }

  logger.info(CH.IMPORT, 'importFromAppState: starting', {
    elements: parsed.elements.length,
    nodes: Object.keys(parsed.nodes ?? {}).length,
  });

  const model = await timeStep(pipelineLog, 'NORMALIZE_STATE', async () => {
    const csvRows = buildUniversalCSV(parsed, { supportMappings: state.sticky?.supportMappings ?? [] });
    const method = state.engineMode === 'common' ? 'ContEngineMethod' : 'Legacy';
    logger.debug(CH.IMPORT, `normalizeToPCF method="${method}" csvRows=${csvRows.length}`);
    const segments = normalizeToPCF(csvRows, { method });
    const nodePos = _resolveNodePositions(csvRows);
    const components = [];

    for (const seg of segments) {
      const type = String(seg.COMPONENT_TYPE || 'PIPE').toUpperCase();
      if (type === 'GHOST' || type === 'MESSAGE-SQUARE') continue;
      const p1 = _pt(seg.EP1) || _pt(nodePos.get(seg.FROM_NODE));
      const p2 = _pt(seg.EP2) || _pt(nodePos.get(seg.TO_NODE));
      const bore = Number(seg.DIAMETER || 0);
      components.push({
        id: seg.REF_NO || `progl-${seg.SEQ_NO}`,
        type,
        ep1: p1,
        ep2: p2,
        bore,
        attributes: {
          'PIPELINE-REFERENCE': seg.PIPELINE_REFERENCE || '',
          MATERIAL: seg.MATERIAL || '',
          SKEY: seg.SKEY || '',
          T1: seg.T1, P1: seg.P1,
          WALL_THICK: seg.WALL_THICK,
          FLUID_DENSITY: seg.FLUID_DENSITY,
          INSUL_DENSITY: seg.INSUL_DENSITY,
          SUPPORT_TAG: seg.SUPPORT_TAG || '',
          SUPPORT_NAME: seg.SUPPORT_NAME || '',
          SUPPORT_DESC: seg.SUPPORT_DESC || '',
          SUPPORT_KIND: seg.SUPPORT_KIND || '',
          SUPPORT_GUID: seg.SUPPORT_GUID || '',
          SUPPORT_FRICTION: seg.SUPPORT_FRICTION,
          SUPPORT_GAP: seg.SUPPORT_GAP,
          SUPPORT_DOFS: seg.SUPPORT_DOFS || '',
          AXIS_COSINES: seg.AXIS_COSINES || '',
          PIPE_AXIS_COSINES: seg.PIPE_AXIS_COSINES || '',
          'COMPONENT-ATTRIBUTE97': seg.REF_NO || '',
          'COMPONENT-ATTRIBUTE98': seg.SEQ_NO || '',
        },
      });
    }
    logger.info(CH.IMPORT, `importFromAppState: built ${components.length} components`);
    return { components };
  });

  const exportScene = await timeStep(pipelineLog, 'BUILD_GEOMETRY',
    async () => buildExportScene(model, pipelineLog));
  const blob = await timeStep(pipelineLog, 'EXPORT_GLB',
    async () => await exportSceneToGLB(exportScene));

  logger.info(CH.IMPORT, `importFromAppState: GLB built`, {
    sizeKb: (blob.size / 1024).toFixed(1),
    components: model.components.length,
  });
  return { blob, componentCount: model.components.length };
}
```

### 5.8 `Proglb_PcfImportAdapter.js` — PCF File → GLB

```javascript
// viewer/js/pcf2glb/import/Proglb_PcfImportAdapter.js
import { parsePcfText } from '../pcf/parsePcfText.js';
import { normalizePcfModel } from '../pcf/normalizePcfModel.js';
import { buildExportScene } from '../glb/buildExportScene.js';
import { exportSceneToGLB } from '../glb/exportSceneToGLB.js';
import { createLogger } from '../debug/logger.js';
import { timeStep } from '../pipeline/timeStep.js';
import { CH } from '../debug/Proglb_DebugChannels.js';

/**
 * Returns { blob, componentCount } or throws.
 */
export async function importFromPcfFile(file, logger) {
  logger.info(CH.IMPORT, `importFromPcfFile: "${file.name}"`, {
    sizeMb: (file.size / 1024 / 1024).toFixed(2),
  });

  const pipelineLog = createLogger();
  // Mirror pipeline log entries to our structured logger
  pipelineLog.subscribe((entry) => {
    const level = entry.level === 'ERROR' ? 'error' : entry.level === 'WARN' ? 'warn' : 'debug';
    logger[level](CH.IMPORT, `[pipeline] ${entry.code}`, entry.data);
  });

  const text = await timeStep(pipelineLog, 'FILE_READ', async () => {
    logger.debug(CH.IMPORT, 'Reading PCF file text...');
    return await file.text();
  });
  logger.debug(CH.IMPORT, `PCF read: ${text.split('\n').length} lines`);

  const parsed = await timeStep(pipelineLog, 'PARSE',
    async () => parsePcfText(text, pipelineLog));
  logger.debug(CH.IMPORT, 'PCF parsed', { blockCount: parsed.blocks?.length });

  const model = await timeStep(pipelineLog, 'NORMALIZE',
    async () => normalizePcfModel(parsed, pipelineLog));
  logger.debug(CH.IMPORT, 'PCF normalized', { componentCount: model.components?.length });

  const exportScene = await timeStep(pipelineLog, 'BUILD_GEOMETRY',
    async () => buildExportScene(model, pipelineLog));

  const blob = await timeStep(pipelineLog, 'EXPORT_GLB',
    async () => await exportSceneToGLB(exportScene));

  logger.info(CH.IMPORT, 'importFromPcfFile: done', {
    components: model.components?.length ?? 0,
    sizeKb: (blob.size / 1024).toFixed(1),
  });
  return { blob, componentCount: model.components?.length ?? 0 };
}
```

```javascript
    pipelineLogEl.style.display = 'block';
    pipelineLogEl.innerHTML = '';

    // Mirror pipeline log to mini log el inside import panel
    const pipelineLog = {
      subscribe: (cb) => {
        _pipelineSubscriber = cb;
      },
    };
    let _pipelineSubscriber = null;
    const pipelineLogProxy = new Proxy(pipelineLog, {
      get: (t, prop) => (...args) => {
        if (_pipelineSubscriber) _pipelineSubscriber({ level: prop, code: args[0], data: args[1] });
        const div = document.createElement('div');
        div.style.color = prop === 'error' ? '#f66' : prop === 'warn' ? '#fa6' : '#669';
        div.textContent = `[${prop}] ${args[0]} ${args[1] ? JSON.stringify(args[1]) : ''}`;
        pipelineLogEl.appendChild(div);
        pipelineLogEl.scrollTop = pipelineLogEl.scrollHeight;
      },
    });

    try {
      let blobUrl, importMeta;

      if (activeSource === 'pcf') {
        progressEl.textContent = 'Parsing PCF...';
        const { blob, componentCount } = await importFromPcfFile(selectedPcfFile, logger);
        blobUrl = URL.createObjectURL(blob);
        importMeta = { source: 'pcf', fileName: selectedPcfFile.name, componentCount };

      } else if (activeSource === 'state') {
        progressEl.textContent = 'Converting app state...';
        const { blob, componentCount } = await importFromAppState(logger, pipelineLogProxy);
        blobUrl = URL.createObjectURL(blob);
        importMeta = { source: 'state', fileName: state.fileName ?? 'app_state', componentCount };

      } else {
        progressEl.textContent = 'Loading GLB...';
        blobUrl = URL.createObjectURL(selectedGlbFile);
        importMeta = { source: 'glb', fileName: selectedGlbFile.name, componentCount: null };
      }

      progressEl.textContent = `✅ Ready — ${importMeta.componentCount ?? '?'} components`;
      logger.info(CH.IMPORT, 'Import complete', importMeta);
      onGlbReady(blobUrl, importMeta);

    } catch (err) {
      progressEl.textContent = `❌ Error: ${err.message}`;
      logger.error(CH.IMPORT, 'Import failed', { message: err.message, stack: err.stack });
      btnLoad.disabled = false;
    }
  });

  _updateLoadButton();

  return {
    destroy: () => {
      logger.debug(CH.IMPORT, 'ImportController destroyed');
    },
  };
}
```

## 8. Property Panel — Per-Component-Type Field Reference

### PIPE
Sections shown: Identification, Geometry, Process Conditions, Pipe Detail

### BEND / ELBOW
Sections shown: Identification, Bend Geometry (with centre point, radius, angle), Process Conditions

### FLANGE
Sections shown: Identification, Geometry, Flange Data (rating, facing, bolt set, gasket, ASME class), Process Conditions

### VALVE
Sections shown: Identification, Geometry, Valve Data (type, rating, body material, actuator, tag no), Process Conditions

### SUPPORT / ANCI
Sections shown: Identification, Location (coords, bore), Support Detail (name, kind, GUID, DOFs, friction, gap, axis cosines)
> Note: process conditions are NOT shown for supports (no T1/P1 meaning)

### REDUCER
Sections shown: Identification, Reducer Geometry (both end bores, concentric vs eccentric), Process Conditions

### TEE / OLET
Sections shown: Identification, Tee Geometry (run ep1/ep2, branch, both bores), Process Conditions

### INSTRUMENT
Sections shown: Identification, Geometry, Instrument Data (tag no, type, connection type), Process Conditions

### Unknown / Default
Sections shown: Identification, Geometry, Process Conditions, Raw Attributes (dynamic dump)

To add a new component type: call `registerSchema('MYTYPE', mySchema)` in
`Proglb_ComponentTypeSchemas.js`. No other code changes required.
